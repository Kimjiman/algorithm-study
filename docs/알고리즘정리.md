# 알고리즘 정리

---

## Hash (해시)

**언제 쓰나**
- 빠른 조회 O(1)이 필요할 때
- 카운팅, 중복 확인, 그룹핑

**핵심 함수**
| 함수 | 설명 |
|------|------|
| `put(k, v)` | 요소 추가/수정 |
| `get(k)` | 값 조회 |
| `getOrDefault(k, default)` | 없으면 기본값 반환 |
| `remove(k)` | 삭제 |
| `containsKey(k)` | 키 존재 여부 |
| `keySet()` | 키 목록 |
| `entrySet()` | 키-값 쌍 목록 |

**자주 쓰는 패턴**
```java
// 카운팅
map.put(key, map.getOrDefault(key, 0) + 1);
```

## Stack (스택)

**개념**: LIFO (Last In, First Out) — 마지막에 넣은 것이 먼저 나온다.

**언제 쓰나**
- 괄호 매칭
- 뒤로가기 기능
- 재귀를 반복문으로 대체할 때

**핵심 함수**
| 함수 | 설명 |
|------|------|
| `push(e)` | 요소 추가 |
| `pop()` | 꺼내기 (제거 + 반환) |
| `peek()` | 꺼내지 않고 맨 위 확인 |
| `isEmpty()` | 비어있는지 확인 |
| `size()` | 크기 확인 |
