# 알고리즘 정리

---

## Hash (해시)

**언제 쓰나**
- 빠른 조회 O(1)이 필요할 때
- 카운팅, 중복 확인, 그룹핑

**핵심 함수**
| 함수 | 설명 |
|------|------|
| `put(k, v)` | 요소 추가/수정 |
| `get(k)` | 값 조회 |
| `getOrDefault(k, default)` | 없으면 기본값 반환 |
| `remove(k)` | 삭제 |
| `containsKey(k)` | 키 존재 여부 |
| `keySet()` | 키 목록 |
| `entrySet()` | 키-값 쌍 목록 |

**자주 쓰는 패턴**
```java
// 카운팅
map.put(key, map.getOrDefault(key, 0) + 1);
```

## Stack (스택)

**개념**: LIFO (Last In, First Out) — 마지막에 넣은 것이 먼저 나온다.

**언제 쓰나**
- 괄호 매칭
- 뒤로가기 기능
- 재귀를 반복문으로 대체할 때

**핵심 함수**

| 함수 | 설명 |
|------|------|
| `push(e)` | 요소 추가 |
| `pop()` | 꺼내기 (제거 + 반환) |
| `peek()` | 꺼내지 않고 맨 위 확인 |
| `isEmpty()` | 비어있는지 확인 |
| `size()` | 크기 확인 |

---

## Queue (큐)

**개념**: FIFO (First In, First Out) — 먼저 넣은 것이 먼저 나온다.

**언제 쓰나**
- BFS (너비 우선 탐색)
- 순서가 중요한 처리 (기능개발, 프로세스 등)

**핵심 함수**

Java에서 큐는 `Queue` 인터페이스 + `LinkedList` 구현체를 주로 사용합니다.

```java
Queue<Integer> queue = new LinkedList<>();
```

| 함수 | 설명 |
|------|------|
| `offer(e)` | 요소 추가 |
| `poll()` | 꺼내기 (제거 + 반환, 없으면 null) |
| `peek()` | 꺼내지 않고 맨 앞 확인 |
| `isEmpty()` | 비어있는지 확인 |
| `size()` | 크기 확인 |

## DFS (깊이 우선 탐색)

**개념**: 한 방향으로 끝까지 가다가, 막히면 되돌아와서 다른 방향 탐색. (백트래킹)

**언제 쓰나**
- 모든 경우의 수 탐색 (순열, 조합)
- 경로 탐색

**재귀 템플릿**
```java
boolean[] visited = new boolean[크기];

void dfs(상태) {
    // 1. 종료 조건
    if (종료 조건) {
        결과 처리;
        return;
    }

    // 2. 선택지 순회
    for (선택지 : 가능한 선택들) {
        if (!visited[선택지]) {
            visited[선택지] = true;   // 선택
            dfs(다음 상태);           // 탐색
            visited[선택지] = false;  // 선택 취소 (백트래킹)
        }
    }
}
```

## BFS (너비 우선 탐색)

**개념**: 가까운 것부터 순서대로 탐색.

**언제 쓰나**
- 최단 거리, 최소 횟수

**큐 템플릿**
```java
Queue<상태> queue = new LinkedList<>();
boolean[] visited = new boolean[크기];

queue.add(시작);
visited[시작] = true;

while (!queue.isEmpty()) {
    상태 cur = queue.poll();

    for (다음 상태 : cur에서 갈 수 있는 곳들) {
        if (!visited[다음]) {
            visited[다음] = true;
            queue.add(다음);
        }
    }
}
```

**핵심 차이**

| | DFS | BFS |
|--|-----|-----|
| 자료구조 | 재귀(스택) | 큐 |
| 용도 | 모든 경우 탐색 | 최단 거리 |
| 키워드 | 순열, 조합, 경우의 수 | 최단, 최소 횟수 |
